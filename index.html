PujanMart

E-Commerce Android App for Pooja Services

×
Project Overview
Screenshots & Video
Details

Overview: PujanMart is an e-commerce Android application designed to facilitate the online booking of pooja services and the purchase of pooja items.

Features:
  • User Registration & Login
  • Browse Pooja Services
  • Book Pooja Online
  • Purchase Pooja Items
  • Shopping Cart
  • Payment Integration
  • Order History
  • Push Notifications
Technologies Used:

Java, Kotlin, Firebase, Payment Gateway Integration, Material Design

My Contribution:
  • Developed the complete frontend architecture using Java & XML following the MVVM pattern.
  • Integrated REST APIs for product listings, pooja bookings, and user management.
  • Built features like Search, Cart, Payment Gateway, and Personalized User Panel.
  • Collaborated with the backend team to bind C# .NET APIs with the Android app.
